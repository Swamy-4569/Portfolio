# Personal Portfolio Website

## Overview
This is a personal portfolio website designed to showcase my skills, projects, and resume. The website is fully responsive and built using HTML, CSS, and JavaScript.

## Live Demo
https://swamy-4569.github.io/Portfolio/

## Features
- Responsive design for all devices
- About section introducing myself
- Projects section showcasing work
- Skills section highlighting technical skills
- Contact section for communication

## Technologies Used
- HTML
- CSS
- JavaScript

## Project Structure
Portfolio/
│── index.html
│── style.css
│── script.js
│── assets/
│── images/

## Setup Instructions

1. Clone the repository
   git clone https://github.com/Swamy-4569/Portfolio.git

2. Navigate to the project folder
   cd Portfolio

3. Open the index.html file in any web browser

## Deployment
This project can be deployed using GitHub Pages or any static hosting service.
